PRを作成します。

## フォーマット

- タイトル: [{チケット番号}] {チケットのタイトル}
  - 例: [FOO-1234] ○○画面の実装
- Assignee: 指示した人
- 概要欄: 以下で固定とする
    ```
    ## チケット

    - x

    ## 仕様書・Figma

    - x

    ## やったこと

    - x

    ## やってないこと

    - x

    ## 参考リンク

    - x
    ```
- 指示がない限りDraftとする
